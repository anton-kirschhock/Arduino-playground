# LED strip controller

Project which changes the colour of a ledstrip using a WPF (.net core) UI.

# Technologies used:
- Arduino Nano
- Generic RGB LED strip (not Addressable)
- WPF (.Net Core)
  - https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit
- Visual Studio 2019
- Visual Studio Code
  - Arduino Plugin
